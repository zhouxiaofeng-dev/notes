所需的npm安装

```javascript
npm i express   //安装express

npm install express-session //安装express-session中间件   用来session身份认证

npm i mysql   //安装mysql

npm i moment   //安装格式化时间包

npm i cors   //解决跨域问题

npm i axios  //安装axios

npm i less  //安装less

npm i less-loader  //安装less-loader  好像和less有版本限制

npm i vuex   //安装vuex

npm install jsonwebtoken express-jwt //安装JWT相关的包 
//jsonwebtoken 生成jwt字符串
//express-jwt 将JWT字符串解析还原成JSON对象 

npm i redux  //安装redux

npm i redux-thunk //安装

npm i react-redux //react内部设计的redux  更加方便
```
